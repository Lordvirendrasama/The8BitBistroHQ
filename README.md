# Pixel Perks: The 8 Bit Bistro Members Club

Pixel Perks is a gamified loyalty and cafe management platform designed specifically for **The 8 Bit Bistro**. It blends retro gaming aesthetics with modern operational tools.

## 🚀 One-Click Deployment (v1.9.4)
To push your latest changes to GitHub, you can now use the automated script:

### Method 1: Terminal (Recommended)
Run this command in your terminal:
```bash
npm run push
```

### Method 2: IDE "Play" Button
If you are using VS Code or a similar editor, open `package.json`, find the `"push"` script under `scripts`, and click the **Play** icon next to it.
